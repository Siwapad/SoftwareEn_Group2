## 1) Overview

ระบบนี้ช่วยให้ผู้ใช้สามารถ

- เข้าสู่ระบบด้วย **รหัสนักศึกษา (มีขีด)** และรหัสผ่าน
- ดู **ตารางสอบ** ของผู้ใช้ที่ล็อกอิน
- ดู **รายการรายวิชาทั้งหมด** (Guest) และใช้ **Search/Filter**
- (ถ้ามีในระบบ) ใช้งานฟีเจอร์เกี่ยวกับ **ลืมรหัสผ่าน/รีเซ็ตรหัสผ่าน** และ **จัดการรูปภาพ/ดาวน์โหลด**

> หมายเหตุ: คู่มือนี้เขียนตามแนวคิดในแผนงาน (Blueprint) ของทีม

---

## 2) User Roles

### 2.1 Guest (ไม่ล็อกอิน)

ทำได้:

- ดูรายการรายวิชาทั้งหมด
- ค้นหา/กรองรายวิชาด้วย รหัสวิชา / ชื่อวิชา / section

ข้อจำกัด:

- ไม่เห็น “ข้อมูลเฉพาะผู้ใช้” เช่น ตารางสอบของตัวเอง

### 2.2 Logged-in User (ล็อกอินแล้ว)

ทำได้เพิ่มจาก Guest:

- ดูตารางสอบของตนเอง (จากรายวิชาที่ลงทะเบียน)
- ใช้งานฟีเจอร์ที่ต้องยืนยันตัวตน เช่น เพิ่มรายการ, ดาวน์โหลด, ฯลฯ

## 3) System Requirements

แนะนำ:

- Browser: Google Chrome หรือ Microsoft Edge เวอร์ชันล่าสุด
- อินเทอร์เน็ตปกติ
- ความละเอียดจอ: ตั้งแต่ 1366×768 ขึ้นไป (มือถือยังใช้งานได้ถ้า UI รองรับ)

---

## 4) Getting Started

1. เปิดเว็บไซต์ของระบบ (URL ตามที่ทีมกำหนด)
2. เลือกใช้งานในโหมด:
   - **Guest**: เข้าดูรายวิชาได้เลย
   - **Login**: เข้าสู่ระบบเพื่อดูข้อมูลส่วนตัว (เช่น ตารางสอบ)

---

## 5) Login (เข้าสู่ระบบ)

### 5.1 วิธีเข้าสู่ระบบ

1. ไปที่หน้า **Login**
2. กรอกข้อมูล:
   - **Username:** รหัสนักศึกษา **ต้องมีขีด (-)**  
     ตัวอย่าง: `663380026-5`
   - **Password:** รหัสผ่านของผู้ใช้
3. กดปุ่ม **Login**
4. หากสำเร็จ ระบบจะพาไปหน้าหลัก/หน้าที่กำหนด และสามารถใช้งานฟีเจอร์ของผู้ใช้ได้

### 5.2 ข้อผิดพลาดที่พบบ่อย

- กรอกรหัสนักศึกษาไม่มีขีด → ระบบจะแจ้งเตือนให้กรอกรูปแบบให้ถูกต้อง
- รหัสผ่านผิด → แจ้งเตือน “Username หรือ Password ไม่ถูกต้อง”
- ช่องว่าง → แจ้งเตือนให้กรอกข้อมูลให้ครบ

---

## 6) Exam Schedule (ตารางสอบ) — สำหรับผู้ใช้ที่ล็อกอิน

### 6.1 ดูตารางสอบ

1. ล็อกอินให้เรียบร้อย
2. ไปที่เมนู **Exam Schedule**
3. ระบบจะแสดงตารางสอบของผู้ใช้ตามรายวิชาที่ลงทะเบียน

ข้อมูลที่มักแสดง:

- รหัสวิชา / ชื่อวิชา
- วันที่สอบ / เวลา
- ห้องสอบ (ถ้ามี)

> หากไม่มีข้อมูลในเทอมนั้น ระบบจะแสดงข้อความ “ไม่พบตารางสอบ”

---

## 7) Subject List (รายการรายวิชา) — Guest & Logged-in

### 7.1 เปิดดูรายการรายวิชา

1. ไปที่หน้า **Subjects**
2. ระบบจะแสดงรายการรายวิชาในรูปแบบตาราง/ลิสต์

ข้อมูลที่มักแสดง:

- รหัสวิชา
- ชื่อวิชา
- Section
- ผู้สอน (ถ้าระบบ join ข้อมูล)

---

## 8) Search & Filter Subjects (ค้นหา/กรองรายวิชา) — Guest

### 8.1 ช่องค้นหา

- **Course Code (รหัสวิชา)**
- **Course Name (ชื่อวิชา)**
- **Section**

> ทุกช่องเป็น “ไม่บังคับ” ผู้ใช้กรอกบางช่องได้

### 8.2 วิธีค้นหา/กรอง

1. กรอกเงื่อนไขที่ต้องการ (กรอก 1 ช่องหรือหลายช่องก็ได้)
2. กดปุ่ม **Search / Filter**
3. ระบบจะแสดงผลลัพธ์ที่ตรงกับเงื่อนไข

### 8.3 กติกาการกรอง (Guest)

- หากกรอกหลายช่องพร้อมกัน ระบบจะใช้เงื่อนไขแบบ **AND**
- รหัสวิชาและชื่อวิชาเป็นแบบ **partial match** (ค้นหาบางส่วนได้)
- Section เป็นแบบ **exact match** (ต้องตรงค่า)

ตัวอย่าง:

- รหัสวิชา = `CS` → แสดงวิชาที่มี `CS` ในรหัส
- ชื่อวิชา = `Data` → แสดงวิชาที่มีคำว่า `Data`
- Section = `1` → แสดงเฉพาะ section 1
- รหัสวิชา = `CS` และ Section = `2` → ต้องตรงทั้งสองเงื่อนไข

### 8.4 กรณีไม่พบข้อมูล

- ระบบจะแสดงข้อความ: **“ไม่พบรายวิชาที่ตรงกับเงื่อนไขการค้นหา”**

### 8.5 การกรอกข้อมูลผิดรูปแบบ

- Section ไม่ใช่ตัวเลข → แจ้งเตือน “Section ต้องเป็นตัวเลข”
- รหัสวิชามีอักขระพิเศษ (เช่น `CS@101`) → แจ้งเตือน “รหัสวิชาห้ามมีอักขระพิเศษ”

## 9) Forgot Password & Reset Password (ถ้ามีในระบบ)

### 9.1 Forgot Password

1. ไปที่หน้า **Forgot Password**
2. กรอก Email ของผู้ใช้
3. กด **Send**
4. ระบบส่งอีเมลสำหรับรีเซ็ตรหัสผ่าน

### 9.2 Reset Password

1. เปิดลิงก์จากอีเมล
2. กำหนดรหัสผ่านใหม่
3. กด **Confirm**
4. ระบบบันทึกรหัสผ่านใหม่และกลับไปหน้า Login

---

## 10) Troubleshooting (แก้ปัญหาเบื้องต้น)

- หน้าไม่แสดงผล → ลองรีเฟรช / เปิดด้วย Chrome
- ค้นหาแล้วไม่เจอ → ตรวจ spelling หรือเงื่อนไขมากเกินไป
- Section ค้นหาไม่ได้ → ตรวจว่ากรอกเป็นตัวเลขเท่านั้น
- ล็อกอินไม่ได้ → ตรวจรูปแบบรหัสนักศึกษา “ต้องมีขีด (-)”

## 11) Limitations

- ฟีเจอร์บางส่วนอาจอยู่ระหว่างพัฒนา (ตามแผนงานของทีม)
- รายการข้อมูลอาจเป็นข้อมูลทดสอบ (dummy data) ขึ้นอยู่กับเวอร์ชันระบบ
